Software Ray Tracer

Made using C++

Plaftorms: Windows

Solo Project

Description

In the second 8-week block at Breda University of Applied Science, I made a software ray tracer in C++.

These are the features I was able to implement in those 8 weeks:

  • Full path-tracing, with optional real-time ray tracing compile flag
  • Rough reflections
  • Anti-aliasing
  • Optimized triangle meshes using Bounding Volume Hierarchy acceleration structures
  • Multi-threaded rendering
  • Textured planes and skybox
